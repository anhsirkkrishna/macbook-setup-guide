MacBook Setup Guide

Unboxing Your Potential

Receiving a new MacBook, especially as you begin your journey in a demanding field like Data Science and AI, is more than just getting a new piece of hardware. It's an opportunity to build a powerful, personalized environment that will support your learning, coding, and creativity for years to come. This guide is designed to be your trusted companion through that initial setup process.

The goal here is not simply to list a series of steps but to help you transform this machine from a generic tool into a bespoke workstation. We will move beyond the defaults, making intentional choices that enhance security, boost productivity, and lay a professional-grade foundation for your coding projects. This process is divided into five key phases: establishing the foundations in the first hour, hardening security, personalizing your workflow, installing an arsenal of essential apps, and finally, building a robust developer environment tailored for Data Science and AI.

The First Hour

Foundational Setup

1.1 The Welcome Mat: Navigating Setup Assistant

When you first open your MacBook, the Setup Assistant will guide you through the basics. It's highly recommended to **set up as a new Mac** instead of using Migration Assistant. A fresh start ensures you don't carry over old files or clutter, resulting in a leaner, faster system from day one.

1.2 System Integrity Check: Updates First, Always

Before anything else, ensure your system is fully updated to patch security risks and prevent compatibility issues.

  • macOS System Updates: Go to Apple Menu → System Settings → General → Software Update.
  • Mac App Store Updates: Open the App Store and click the Updates tab.

1.3 Digital Declutter: Creating a Clean Slate

Remove pre-installed apps you won't use (like GarageBand or iMovie) to free up space. Open Finder → Applications, right-click an app, and select Move to Bin. Empty the Bin to finalize.

Fort Knox Foundation

Essential Security & Privacy

2.1 Login & Access Control

Use a strong, unique password and set up Touch ID. Set your Mac to lock immediately upon sleep in System Settings → Lock Screen.

2.2 Data Encryption with FileVault

Enable full-disk encryption to protect your data if your Mac is lost or stolen. Go to System Settings → Privacy & Security → FileVault and turn it on. Using your iCloud account for recovery is the most convenient option.

2.3 Configure the macOS Firewall

The firewall is off by default. Enable it for crucial protection on public Wi-Fi. Go to System Settings → Network → Firewall and turn it on. Consider enabling "Stealth Mode" in the options.

2.4 Master App Permissions

Review and manage app permissions in System Settings → Privacy & Security. Grant apps only the permissions they absolutely need. Ensure "Allow applications downloaded from" is set to "App Store and identified developers."

2.5 The Safety Net: Time Machine

Regular backups are non-negotiable. Use an external drive (at least 2x your Mac's storage) to set up Time Machine for automatic, hourly backups. Connect the drive and follow the prompts, or configure it in System Settings → General → Time Machine.

Making It Yours

Personalization & Workflow

3.1 Mastering the Magic Trackpad

In System Settings → Trackpad, enable "Tap to click" and increase the "Tracking speed" for a faster, more fluid experience. Learn these core gestures:

Gesture Action
Three-finger swipe up Open Mission Control to see all open windows.
Three-finger swipe left/right Switch between full-screen apps and desktops.
Pinch with thumb & three fingers Show Launchpad to see all your apps.

3.2 Taming the Dock & Menu Bar

Keep your Dock clean with only essential apps. In System Settings → Desktop & Dock, enable "Automatically hide and show the Dock" and disable "Show recent applications in Dock" to maximize screen space.

3.3 Finder Finesse

Optimize Finder for a developer workflow:

  • In Finder → Settings → General, set "New Finder windows show:" to your Home Folder.
  • In the View menu, select Show Path Bar and Show Status Bar for more context.
  • Set your preferred view (Columns view is great for developers) and click "Use as Defaults" in View Options.

The App Arsenal

Essential Software

Google Chrome

A practical choice for its extension library and seamless integration with Google services.

Notion / Obsidian

Powerful "second brain" apps for organizing notes, tasks, and knowledge. Notion is an all-in-one workspace, while Obsidian excels at linked, local-first notes.

Rectangle

A free, essential utility that brings window snapping functionality to macOS for easy multitasking.

Raycast / Alfred

Supercharged application launchers that replace Spotlight with powerful workflows, calculations, and clipboard history.

Bitwarden

A free, open-source, and cross-platform password manager to keep your accounts secure everywhere.

CleanShot X

A massive upgrade to the default screenshot tool, with scrolling captures, advanced annotation, and cloud sharing.

The Coder's Launchpad

A Foundation for DS & AI

5.1 Prerequisite: Xcode Command Line Tools

This package includes essential tools like Git and compilers needed for development. Install it by running this command in the Terminal app:

xcode-select --install

5.2 Package Manager: Homebrew

Homebrew is "The Missing Package Manager for macOS," used to install thousands of open-source tools. Install it with:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Critical: On Apple Silicon (M1/M2/M3) Macs, you must run the two "Next steps" commands shown at the end of the installation to add Homebrew to your PATH.

5.3 Modern Terminal: iTerm2 & Oh My Zsh

Upgrade the default Terminal with iTerm2 for features like split panes and better search. Then, enhance your shell with Oh My Zsh for powerful plugins and themes.

Install iTerm2:

brew install --cask iterm2

Install Oh My Zsh:

sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

5.4 Core Language: Python with Miniconda

Do not use the system Python. The industry standard is to use a dedicated distribution like Miniconda to manage isolated project environments. Install it with:

curl -sL "https://repo.anaconda.com/miniconda/Miniconda3-latest-MacOSX-arm64.sh" > "Miniconda3.sh"
bash Miniconda3.sh

After installation, close and reopen your terminal. Create and activate a project environment with: 

conda create -n ds_project python=3.10 pandas numpy jupyter
conda activate ds_project

5.5 Code Editor: Visual Studio Code

VS Code is the standard for a reason. Install it and its essential Python extensions:

brew install --cask visual-studio-code

Inside VS Code, install the Python and Jupyter extensions from the marketplace. Then, use the Command Palette (Cmd+Shift+P) to run `Python: Select Interpreter` and choose your `ds_project` conda environment.
